Create Tournament Bracket

This randomly seeds all registered teams into a double elimination bracket. The bracket size is automatically chosen based on team count (8, 16, 32, 64, or 128). Byes are added automatically for empty slots. This action cannot be undone — registration will close.

Reset tournament: Deletes all matches and reopens registration. Use only if starting over.

Create Pool Play Schedule

Randomly splits all registered teams into Pool A and Pool B, then generates a round-robin schedule within each pool using the circle method. With 26 teams (13 per pool) and 6 courts per pool, 6 rounds guarantees every team plays 5–6 games. This closes registration.
